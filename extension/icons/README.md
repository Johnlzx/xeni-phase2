# Form Pilot Extension Icons

To complete the extension setup, add PNG icons in the following sizes:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

Recommended design:
- Background: #0E4268 (Xeni brand blue)
- Foreground: White "FP" text or play icon
- Rounded corners

You can generate these icons using any image editor or online tool.
For quick testing, the extension will work with placeholder icons.
